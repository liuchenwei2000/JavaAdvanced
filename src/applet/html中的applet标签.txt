                       Html嵌入Applet的问题
标签<Applet>的属性介绍
　属性    　　　　含义
　Width    　Applet在Html页面上的宽度

　Height    　Applet在Html页面上的高度

　Name    　Applet在Html页面上的名称,用于区名一个Html页面上的多个Applet

　Code    　Applet类名(含包名)，必须带后缀"class"
这个属性给出applet类文件的名称，这个名称不是相对于codebase的，就是在没有指定codebase的时候，相对于当前页面的
路径名必须和applet类的包相匹配，如"com.Applet.class"，但是在这里不能使用绝对路径名
如果类文件存放在其他地方，那么就应该使用codebase属性，code属性仅仅用于指定包含applet类的类名

　Codebase    Applet的类相对路径，相对于Html页面位置
这个可选属性指出用于定位类文件的URL，可以使用绝对URL
比如html文件和applet文件夹在同一个目录下，并且applet文件夹中含有Applet.class类文件
那么可以这样标记html：
<applet code = "Applet.class" codebase = "applet"...>

　Archive    　Applet所在Jar包的文件名

　标签<PARAM>的属性介绍
　属性    　　　　含义
　Name    　Applet传入的参数名
　Value    　Applet传入的参数值
